# Can We Trust AI? - Interactive Simulation Website

An educational website for 6th grade students to learn about AI trustworthiness, information integrity, and cybersecurity.

## 🎓 Lessons Included

1. **Lesson 1: Can We Trust AI?**
   - Explore AI-generated images with flaws
   - Understand that AI can make mistakes
   - Evaluate AI-generated information

2. **Lesson 2: How Does AI See Images?**
   - Convert images to data
   - Learn about features and patterns
   - Understand missing information problems

3. **Lesson 4: Can We Trust Information?**
   - See how data changes during transmission
   - Understand message tampering
   - Learn why security matters

4. **Lesson 5: How Can We Help AI?**
   - Full Edge AI simulation
   - Build secure systems
   - Understand encryption

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
git clone https://github.com/staciasuarez01-del/AI.SIM.git
cd AI.SIM
npm install
npm run dev
```

The site will open at `http://localhost:3000`

### Building for Production

```bash
npm run build
```
